## Qanday ishlatish

### 1. Loyihani yuklab olish

```bash
git clone https://github.com/Dilshodbek-Ilhomov/form_site_main.git
cd form_site_main
```

### 2. Kerakli kutubxonalarni o‘rnatish

```bash
pip install -r requirements.txt
```

### 3. Loyihani ishga tushirish

Loyihani **Django** asosida ishlatish:

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

Brauzer orqali ochamiz:

```
http://127.0.0.1:8000/
```
Loyiha ishga tushgandan so‘ng asosiy funksiyalar avtomatik tarzda ishlay boshlaydi.
